# Student Performance Analysis Dashboard

## Project Overview

The **Student Performance Analysis Dashboard** is a data analytics and visualization project developed to analyze student academic performance and identify patterns that influence learning outcomes.

The project transforms raw student data into meaningful insights using data cleaning, analysis, and interactive visualization techniques. The final dashboard was created using Microsoft Power BI to help users understand performance trends and make data-driven decisions.

---

## Project Objective

The main objectives of this project are:

* Analyze student academic performance patterns
* Identify factors affecting student results
* Understand subject-wise performance trends
* Create an interactive dashboard for effective data visualization
* Generate meaningful insights from educational data

---

## Dataset Information

**Dataset Source:** Kaggle

**Dataset Type:** Student Performance Dataset

The dataset contains information related to students, including academic scores and other attributes useful for analyzing performance trends.

## Project Workflow

### 1. Data Collection

The dataset was collected from Kaggle and examined to understand the available attributes and structure.

### 2. Data Cleaning & Preprocessing

The raw dataset was processed to improve data quality by:

* Handling missing values
* Removing inconsistencies
* Formatting data fields
* Preparing the dataset for visualization and analysis

### 3. Data Analysis & Visualization

The cleaned dataset was imported into Power BI to create an interactive dashboard containing charts, KPIs, and filters for performance analysis.

---

## Dashboard Features

The dashboard provides insights into:

* Overall student performance
* Subject-wise performance analysis
* Performance distribution patterns
* Comparison of student results
* Identification of important performance trends
