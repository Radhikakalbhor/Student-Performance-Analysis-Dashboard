**PROJECT SUMMARY**



**Project Title:**

Student Performance Analysis Dashboard





**1. Project Objective:**

The objective of this project is to analyze student academic performance and identify factors affecting students' results. The dashboard provides insights into marks distribution, subject-wise performance, and student-related factors to support data-driven decision-making.





**2. Dataset Source:**

Dataset obtained from Kaggle.



**Dataset Type:**

Student Performance Dataset



The dataset contains information related to student demographics, academic performance, and other attributes useful for analyzing factors influencing student outcomes.





**3. Tools Used:**

\- Microsoft Power BI (Data Visualization and Dashboard Creation)

\- Power Query (Data Cleaning and Transformation)

\- Microsoft Excel / CSV (Data Handling)

\- DAX (Data Analysis Expressions for Calculations)





**4. Key Features:**

\- Data cleaning and preprocessing

\- Interactive Power BI dashboard

\- Analysis of student performance trends

\- Subject-wise performance analysis

\- Identification of factors affecting academic results

\- Visual representation using charts, graphs, and KPIs

\- Data-driven insights for improving student outcomes





**5. Project Outcome:**

The project provides meaningful insights into student performance patterns and demonstrates the application of data analytics techniques for educational decision-making.
